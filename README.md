# archagents

Monorepo for AI agent projects.

## Projects

### [SENTINEL](sentinel/)

AI-Generated Code Governance & Compliance Platform for regulated industries. Event-driven architecture with 7 analysis agents, HMAC-signed compliance certificates, and enterprise security features (EU AI Act, SOC 2, ISO 27001).

- **TypeScript**: API server, CLI, Next.js dashboard, event bus, assessor
- **Python**: Security, license, dependency, AI-detection, quality, policy, and LLM review agents
- **604 tests** across both stacks

See [`sentinel/README.md`](sentinel/README.md) for full documentation.
